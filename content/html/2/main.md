# Una página web

Una página web es un archivo HTML. Puedes escribir HTML 
en cualquier archivo con la extención .html o .htm 
usando un editor de código como 
[**Visual Studio Code**](https://code.visualstudio.com/) o 
[**Spck Editor**](spck.io)
